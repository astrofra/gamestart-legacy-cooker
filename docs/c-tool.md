# Native C Tool

The repository includes a standalone C implementation at:

```text
c/gamestart_cooker.c
```

It implements the same commands as the Python tool:

```powershell
gamestart-cooker-c info game.nac
gamestart-cooker-c list --include "data/music/*" game.nac
gamestart-cooker-c unpack game.nac unpacked --overwrite
gamestart-cooker-c pack unpacked repacked.nac --compression 6 --raw "data/sfx/*" --raw "data/music/*" --overwrite
```

## Build With CMake

```powershell
cmake -S . -B build
cmake --build build --config Release
```

The executable will be generated by the selected CMake generator, usually under
`build\Release\gamestart-cooker-c.exe` on Visual Studio generators or
`build\gamestart-cooker-c.exe` with single-configuration generators.

## Direct Compiler Commands

MSVC developer shell:

```powershell
cl /O2 /MT /I third_party\zlib c\gamestart_cooker.c third_party\zlib\adler32.c third_party\zlib\compress.c third_party\zlib\crc32.c third_party\zlib\deflate.c third_party\zlib\inffast.c third_party\zlib\inflate.c third_party\zlib\inftrees.c third_party\zlib\trees.c third_party\zlib\uncompr.c third_party\zlib\zutil.c /Fe:gamestart-cooker-c.exe
```

GCC or Clang:

```sh
cc -O2 -Ithird_party/zlib -o gamestart-cooker-c c/gamestart_cooker.c third_party/zlib/adler32.c third_party/zlib/compress.c third_party/zlib/crc32.c third_party/zlib/deflate.c third_party/zlib/inffast.c third_party/zlib/inflate.c third_party/zlib/inftrees.c third_party/zlib/trees.c third_party/zlib/uncompr.c third_party/zlib/zutil.c
```

## Dependencies

The C tool vendors the zlib source files it needs in `third_party/zlib`, so it
does not need an external zlib installation.

## Notes

- Paths inside the archive use forward slashes.
- Extraction rejects absolute paths, drive-qualified paths, and `..`
  traversal.
- `--raw` can be repeated for files that should stay uncompressed, such as
  Magnetis audio entries.
